# 登入系統使用說明

## 系統架構

已成功建立完整的登入系統，包含以下三個表單：

### 1. LoginForm（登入表單）
**檔案：** `LoginForm.cs`

**功能：**
- 提供帳號和密碼輸入欄位
- 密碼預設以 `*` 字元顯示
- 「顯示密碼」勾選框可切換密碼顯示/隱藏
- 「登入」按鈕驗證帳號密碼
- 「忘記密碼?」按鈕開啟重設密碼表單

**預設帳號密碼：**
- 帳號：`admin`
- 密碼：`1234`

**操作說明：**
1. 輸入帳號和密碼
2. 可勾選「顯示密碼」查看輸入內容
3. 點擊「登入」或按 Enter 鍵登入
4. 登入成功後進入主系統（Form1）
5. 登入失敗會顯示錯誤訊息表單

### 2. ResetPasswordForm（重設密碼表單）
**檔案：** `ResetPasswordForm.cs`

**功能：**
- 重設新的帳號和密碼
- 需輸入新帳號、新密碼和確認密碼
- 密碼需輸入兩次以確保正確
- 設定後會覆蓋原有的登入資訊

**驗證規則：**
- 帳號長度至少 3 個字元
- 密碼長度至少 4 個字元
- 兩次密碼輸入必須一致
- 確認後會顯示成功訊息

**操作說明：**
1. 在登入畫面點擊「忘記密碼?」
2. 輸入新帳號（至少3個字元）
3. 輸入新密碼（至少4個字元）
4. 再次輸入密碼確認
5. 點擊「確認重設」完成設定
6. 系統會顯示重設成功訊息
7. 使用新帳號密碼登入

### 3. ErrorMessageForm（錯誤訊息表單）
**檔案：** `ErrorMessageForm.cs`

**功能：**
- 顯示登入失敗訊息
- 使用紅色 ? 圖示表示錯誤
- 提供友善的錯誤提示

**觸發時機：**
- 帳號或密碼為空
- 帳號或密碼輸入錯誤

## 程式流程

```
啟動程式
   ↓
顯示 LoginForm（登入表單）
   ↓
   ├─→ 登入成功 → 進入 Form1（主系統）
   │
   ├─→ 登入失敗 → 顯示 ErrorMessageForm（錯誤訊息）
   │                     ↓
   │                 返回登入畫面
   │
   └─→ 點擊「忘記密碼?」→ 開啟 ResetPasswordForm（重設密碼）
                             ↓
                          ├─→ 確認重設 → 更新帳號密碼 → 返回登入畫面
                          │
                          └─→ 取消 → 返回登入畫面
```

## 程式碼修改說明

### Program.cs
修改了主程式入口點，現在會先顯示登入畫面：
```csharp
LoginForm loginForm = new LoginForm();
if (loginForm.ShowDialog() == DialogResult.OK)
{
    // 登入成功，開啟主表單
    Application.Run(new Form1());
}
```

## UI 設計特色

### 登入表單
- 現代化扁平設計風格
- 藍色主題色系（#007ACC）
- 置中顯示，固定大小（450×550）
- 圓角按鈕，無邊框設計
- 支援鍵盤 Enter 快速登入

### 重設密碼表單
- 綠色主題色系（#2E7D32）
- 雙密碼輸入確認機制
- 即時驗證輸入格式
- 詳細的錯誤提示

### 錯誤訊息表單
- 紅色警示風格（#CC0000）
- 大型 ? 錯誤圖示
- 簡潔明瞭的錯誤訊息
- 自動聚焦「確定」按鈕

## 安全特性

1. **密碼保護**
   - 預設使用 `*` 字元遮蔽密碼
   - 提供顯示/隱藏密碼功能

2. **輸入驗證**
   - 防止空白輸入
   - 最小長度限制
   - 密碼二次確認

3. **錯誤處理**
   - 友善的錯誤提示
   - 不顯示具體錯誤原因（安全考量）

## 測試建議

1. **正常登入測試**
   - 使用預設帳號密碼 (admin/1234) 登入

2. **錯誤處理測試**
   - 輸入錯誤的帳號或密碼
   - 留空帳號或密碼欄位

3. **重設密碼測試**
   - 重設為新的帳號密碼
   - 使用新帳號密碼登入

4. **密碼顯示功能測試**
   - 勾選「顯示密碼」查看輸入
   - 取消勾選隱藏密碼

## 擴展建議

如需進一步強化系統，可考慮：
- 連接資料庫儲存帳號密碼
- 加入密碼加密功能
- 實作登入次數限制
- 加入驗證碼功能
- 記住帳號功能
- 自動登入功能
